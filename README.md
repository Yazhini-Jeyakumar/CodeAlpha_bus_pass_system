# Online Bus Ticket Booking System

A cloud-ready online bus ticket booking system built with Flask. The system helps prevent ticket loss, ticket misuse, and incorrect pricing while supporting scalable deployment.

## Features

- User registration and login
- Browse available bus routes
- Seat selection with live availability
- Server-side fare calculation
- Mock payment process
- QR code-based e-tickets
- Downloadable PDF tickets
- Route search by source and destination
- Admin dashboard for managing routes and bookings
- Revenue statistics for administrators

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- Jinja2
- Bootstrap 5
- ReportLab
- qrcode
- Pillow
